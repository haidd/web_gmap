A Map for Partners

The Simple Google Maps module, using the address of partner for displaying!

Using module base_geolocalize to get latitude, longitude and drawing a map with Google Maps JavaScript API.

The Google Maps JavaScript API has the following limits. Users of the standard API has free until exceeding 25,000 map loads per 24 hours. More detail

How to config

Go to Settings > Technical > Parameters > System Parameters

Map URL

The value of URL will call library of the The Google Maps JavaScript API. Default is: https://maps.googleapis.com/maps/api/js

API Key

Get it from Google Map API Key

How to use

Go to contacts > Choose your Partner > Move on page `Partner Assignation`

Geo Localize

Get date, latitude and longitude.

Show map

In case you didn't click button `Geo Localize` before, just click this one. It will get date, latitude and longitude if they don't exist and then show the partner's map.
